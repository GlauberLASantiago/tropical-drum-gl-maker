# Tropical Drum GL (Groove Lab) Maker

Sequenciador visual em HTML para criar ritmos de bateria e exportar padrões compatíveis com o plugin **Tropical Drum GL (Groove Lab)** no MuseScore.

## Recursos

- Criação e edição visual de ritmos em grade
- Presets de ritmos brasileiros e internacionais
- Ajuste de compassos, fórmula de compasso e resolução
- Importação e exportação de padrões em `.qml`
- Cópia rápida do código gerado

## Como usar

1. Abra o arquivo `/home/runner/work/tropical-drum-gl-maker/tropical-drum-gl-maker/index.html` em um navegador.
2. Escolha um ritmo existente ou crie um novo.
3. Edite a grade do sequenciador clicando nas células.
4. Use os botões de importação, cópia ou download para gerar o conteúdo desejado.

## Estrutura do repositório

- `index.html`: aplicação completa com interface, estilos e lógica JavaScript.

## Observações

- O projeto não depende de etapas de build ou instalação.
- Para melhores resultados, use um navegador moderno.
